青岛龙湖·春江天越，这样一个平面图依旧是没有想法怎样具体实施的。注意道路明显宽度有很大的不同，也就是说这些道路应该都是knift project投射的绘制好的平面，而不是用bezier曲线放样一个profile 道路mesh。

配合这个成都的效果图、实景，可以确定道路高差层叠、地砖样式、绿化带之间没有curb。

水体的高差与道路在这个案例中推测为台阶的整体上下，而在泰欣嘉园水体是台阶，道路则是曲线渐进。

box plant和草坪构成与道路的高差，没有curb不过有凹陷的黑色cobblestone。

宁波·誉澜颂，在这个项目中同样也可以发现道路没法用bezier曲线+放样的方式去完成。

从这张图来看水景、道路、绿化带都是带有规律几何特征的不规则形状平面，那么如何去绘制这些平面呢？要知道道路虽然有不规则形状、长宽，但是主线是等宽的，这就意味着主线部分必然是优先精确绘制的，而不可能是通过所有绿化确认后剩余部分，反过来说在道路的面片确认后剩余的绿化、水体倒是可以顺带一起确认。从这个项目来看建筑所占据的平面四周的绿化与其保持法线方向一致的矩形绿化带，也就是说第一步确认建筑的摆放以后要预留一部分绿化带，剩下的部分是主路切分、环绕景观，不过这并不是绝对的，在青岛龙湖·春江天越项目中建筑四周一圈的绿化总体是按着主路的走势。

blender中绘制一条曲线开启curve tools插件进入edit模式，n键进入curve edit插件outline可以对绘制曲线进行“路面扩展”获得等宽间距的道路，复制曲线移动是做不出这种效果的如图3.得到等宽曲线后转换为物体，edge bridge即可连接成道路面片。下面这个视频还演示了插件可以做曲线布尔运算获得异形规则曲线。有了这个工具，我想应该是先绘制主路保证主路等宽

Blender bezier boolean functions #shorts

根据平面参考图绘制主路，可以看到在形成mesh后extrude是没办法延伸“平行曲线”的，但是ctrl r加环形边可以，也就是说只能在内部切分不能外延，像道路带有两侧复合花砖、凹陷+cobblestone都应该绘制整体最大宽度然后切分

plugin中激活add curve : extra objects可以直接创建多边形的Bezier曲线，我试了下从plane转换矩形的曲线用curve boolean无法得到如下图的组合曲线只有bezier创建的彩可以，至于拱形的圆角在curve tool中chamfer、fillet可以对两侧顶点产生圆角、倒角。

宁波中海·江逸源境府 / 澜道设计

从这个角度可以看出来外围绿化带是大量不规则的fillet曲面布尔，似乎没有等宽主路了因此可以确认绿化带从而带出地砖。

小区里的主路是大多有等距的先画主路，绿化带的起伏直接edge loop做个bevel倒角出起伏。

这里的道路作为四条主路的岔口是两个圆叠加，curve boolean即可，2d curve boolean经常会出现死机，所以制作时记得经常保存，曲线主路做一次outline之后f 键make segment之后即成曲面可以与其他曲面curve boolean合并。我在做图3的方圆交集、并集曲线布尔时第一次卡了，第二次同样的操作没卡，暂时不知道什么原因。

道路岔口的弧度通过选中交点fillet产生圆角即可。似乎不存在不规则曲面的直接绘制，比如下图1中3岔路口看起来很奇怪,但总的来说应该类似图4、5这样两两布尔之后形成，这个方法制作岔口很爽没有一点毛病。

甚至可以inset整个面片形成贯穿整体的curb。

儿童乐园什么的基本就是一个面片给个材质而已，至于其上的道具如果是脱离于地形单独建模以后导入的要考虑地形吸附问题。

九江中辉半山墅 / 贝尔高林国际

别墅区它这个相当于在等距主路上切分零碎绿化带。高层公寓部分景观和上两个项目没有太大区别，只不过这种多色花砖看起来不像一张贴图直接盖一个大的面片好解决的，分别率不太好搞，如果是局部切分面片感觉工作量略大。

重庆中海寰宇时代 / 蓝调国际

这个项目涉及一个跳空平台与登上它的楼梯，这部分暂且认为是独立模型后期加上去的。

曲面道路的复合材质，相当于在一个面片上投射曲面分割？由于和原曲面不是edge loop关系，这个曲面要如何添加上去呢？选中要被投射的面片进入面模式，然后加选那个投射物体，knife project即可投射曲面。

相邻的阶梯是在地形整体做完以后独立添加还是制作地形阶段就划分好这些阶梯的曲面体块？

这个项目实景看着很抽象，这个水景感觉要溢出来了。图2更是看不懂水面在哪儿了。整体来看，如果前面3个项目是一个类型，那么它算另一类。

整个道路看起来方方正正。

下1看起来很中海阅湘台。

项目搞下来大概有10几个人。

济南保利·珑誉 / 蓝调国际

这个项目和南京金基·瞻月府的风格类似，从展示的照片来看所有的元素切分都是方方正正的，我突然想起来这俩项目甚至都没看到住宅楼在哪里，保利·珑誉是高层，从房产销售的网站来看完全看不出这个景观也是很诡异。

成都天投中海·天府合印

和重庆中海寰宇时代一样有个高差平台

进小区的大门非常大是一个超大开间的“亭子”，出来拐弯前方是个水景加“屏风”。水景上有个简单的几何雕塑，水景与道路间有下凹的cobblestone。

这个介于上面1类二类风格之间，道路是方方正正的，绿化带偏向1类大块面；2类则是细粒度分割，大多带有整体花坛。1类的地砖色彩较为丰富；2类色彩单一在材质上做分层。花砖可以看出来道路需要局部对uv做调整以适应砖块排布走向。

存在一些曲折设计，将一个较大的地块作为一处小景，这种对一个大块面切分曲面的设计或许可以交给人工智能批量出类似方案？

长沙保利 · 梅溪天珺

这个项目和济南保利珑誉的面积差不多都是7000-8000平方米的景观设计。大门一样是大开间的亭子。不过两个项目似乎都是在建状态并不能看到完整效果。

看来小区大门进去是个品字形的布局包围一个水景。

这个项目是4000多平米，入大门进来是直角曲折，带有栅格顶，四周的绿化看起来层次很丰富，不过比较廉价。

整体上有点凉亭、小游乐场的感觉。

项目6000平米左右，这个项目可以说是第三种风格？它的绿化十分单一反而产生了很鲜明的层次，暂时说不上来这种简洁。我感觉不错。

这个似乎是入户门？这片看起来很不好，很呆板突兀、空旷。提出不好是改进的开端，可以找类似的项目去比较。

成都中海·天府里·领峯

没搞懂，这个只有3栋房子的项目倒是有8000多平米景观设计？

好像它的核心景观是三栋房子所围的三角形中心做了一圈椭圆景观。

它显然属于1类风格，不过在布局上更为工整而没有十分“娇柔做作”式的切分诡异的曲面形状。道路方面复合材质需要看起来是可以edge loop的等距规则主路，只不过这个edge loop要精确计算切分。

长沙建发·养云

这个大门搞了徽派建筑的那种高墙，外面看过去很单调，近看式有纹理的石材，但总体还是单调，迎客松显得也很苍白。墙脊式铝合金，正门前面有个狐假虎威的桥，由于比例尺度问题反而觉得小家子气。

进门后从者个角度看很有气势，这个狮子雕塑似乎是铜？浮雕一般般。

这个似乎是回廊？感觉上很art deco，屋架还用了早期阿拉伯的叶饰雕花。

这里的门甚至有多重葱形拱。

看起来像开元酒店那种多进院落。然而实际上整体是大平层项目有8栋高层，项目25年交付，很难想象这样的景观和高层混合是什么效果。

目前的分析基于园景人1-4页的项目，这些项目都是近期开盘的，从网上看不到整体落地效果，我想翻翻前面几页的项目，或许从150页开始？
